🤖 Android — Chrome (Detected)

🍎 iPhone / iPad (Detected)
Step 1 — Open this app in Safari (not Chrome)
Step 2 — Tap the Share button ⬆ at the bottom
Step 3 — Scroll down and tap "Add to Home Screen"

💻 Laptop / Desktop — Chrome
Step 1 — Open this app in Google Chrome
Step 2 — Look for the install icon ⊕ in the address bar (right side)
Step 3 — Click it and select "Install" ✅
⚠️ The install icon only appears when you open the file via a web server or hosting. If you double-click the file directly, use the step below.

1. Go to github.com → Sign up free

3. Go to Settings → Pages → Enable GitHub Pages
4. Open the generated URL in Chrome → Install icon appears!
